## 编译

项目可以通过两种方式构建

### 1. Visual Studio 2022及以上版本

核心文件: Project1.cpp

编译器: MSVC

### 2. 使用Make构建

> [!note]
> 需要切换到`yumo`分支

核心文件：src/main.cpp

编译命令：在根目录下使用`make`命令（需要安装make4.4版本）

编译器：g++、windres

## 技术栈

- WIN32
- [EasyX](https://easyx.cn/)（仅限图形库的HSLtoRGB库，yumo分支不使用）
